# Weapons Examples

Use these as a starting point for your own weapons.

Some weapons require a minimum stat to use effectively. A Hero using a weapon below its requirement means **disadvantage** — The Hero rolls the d20 twice and take the lower result.

Melee = STR

Ranged = DEX

Magic = MND

---

| Weapon | Damage | Requirement | Type |
|---|---|---|---|
| Bare Hands | 1 | None | Melee |
| Wand | 1 | None | Melee |
| Staff | d4 | None | Magic |
| Dagger | d4 | None | Melee |
| Sling | d4 | None | Ranged | 
| Shortsword | d6 | STR 3+ | Melee |
| Hand axe | d6 | STR 3+ | Melee |
| Shortbow | d6 | DEX 3+ | Ranged |
| Longsword | d8 | STR 4+ | Melee |
| Mace | d8 | STR 4+ | Melee |
| Longbow | d8 | DEX 4+ | Ranged |
| Greatsword | d10 | STR 5+ | Melee |
| Warhammer | d10 | STR 5+ | Melee |
| Heavy crossbow | d10 | DEX 5+ | Ranged |
